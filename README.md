# Brasil Play — Repositório Kodi

Repositório oficial do addon **Brasil Play** para Kodi.

## Como instalar

### 1. Adicionar fonte no Kodi
- Vá em **Configurações → Gerenciador de Arquivos → Adicionar Fonte**
- URL: `https://manotvch-ship-it.github.io/repository.brasil-play`
- Nome: `Brasil Play`

### 2. Instalar o repositório
- Vá em **Add-ons → Instalar de arquivo zip**
- Selecione a fonte `Brasil Play`
- Instale `repository.brasil-play/repository.brasil-play-1.0.0.zip`

### 3. Instalar o addon
- Vá em **Add-ons → Instalar do repositório → Brasil Play Repository**
- Clique em **Brasil Play** e instale

### Atualizações automáticas
O Kodi verificará automaticamente por novas versões. Quando você publicar uma atualização no repositório, todos os usuários serão notificados.
